## 模式研究

### 浮点数匹配

```python
r'-?(\d+(?:\.\d+)?(?:e-?\d+)?)'
```

这个模式能够匹配所有的数字模式，包括正负数，科学计数法模式。这里注意一下`(?:)`这个模式，代表这个括号标出的元组不参与编号，在输
输出的结果中也不会单列出来。

### 汉字

```python
r'^[\u4e00-\u9fa5]{0,}$ '
```

### 邮箱

```python
r'^\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*$ '
```

### 手机号码

```python
r'^(13[0-9]|14[5|7]|15[0|1|2|3|5|6|7|8|9]|18[0|1|2|3|5|6|7|8|9])\d{8}$'
```
